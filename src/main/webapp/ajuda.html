GERCI - Gerênciador de Coordendas Itaipu

O sistema GERCI realiza a conversão entre sistema de coordenadas, e para isso faz a leitura de arquivo contendo os pontos a serem transformados. Para a utilização do sistema é necessário ter em mão os arquivos com os dados referentes aos pontos de controle e com os pontos a serem transformados. Os pontos de controle são os pontos conhecidos nos dois sistemas de coordenadas em que se deseja trabalhar, ou seja, o sistema de origem e o sistema de destino.

O primeiro arquivo a ser carregado no GERCI é um arquivo contendo os pontos de controle. Na janela inicial do sistema clique no botão Escolher Arquivo (ou Choose File, em inglês), dentro do campo Carregar Pontos de Controle. Ao clicar neste botão será aberto uma janela para que o arquivo de entrada, com os pontos de controle, seja selecionado. Após selecionar o aquivo desejado Abrir (ou Open, em inglês).

Quando o arquivo for carregado o sistema irá automaticamente fazer a leitura do arquivo, e irá apresentar os pontos de controle em forma de tabela para que o usuário veja o resultado.

O arquivo com as informações de pontos de controle devem seguir um padrão pré-definido, que pode ser encontrado na página wiki do Celtab: http://wiki.celtab.org.br/index.php/gerci. Exemplos de arquivos de entrada podem ser baixados através dos links que se encontram ao final desta página.

Quando um arquivo não apresenta a estrutura desejada ou contém informações duplicadas, o sistema irá informar ao usuário do problema e reverter a janela para o estado inicial.

Se o processamento do arquivo ocorrer sem nenhum problema os dados serão apresentados e será apresentada um novo campo, chamado Carregar Pontos a Transformar, e será aberta a opção para carregar o arquivo com os dados a serem transformados.

No novo campo aberto clique no botão Escolher Arquivo (ou Choose File, em inglês). Ao clicar neste botão será aberto uma janela para que o arquivo de entrada, com os pontos a transformar, seja selecionado. Após selecionar o aquivo desejado Abrir (ou Open, em inglês).

Após o arquivo ser carregado o sistema irá automaticamente realizar a transformação entre os sistemas de coordenadas. Nesta etapa, se o arquivo com os pontos a transformar contiverem pontos iguais, estes serão indicados com uma linha vermelha na tabela que será apresentada.

Após a realização da transformação será apresentado dois botões:
Salvar em Texto: Salva o resultado da transformação em um arquivo de texto.
Nova Transformação: Volta a janela principal para o estado inicial.

Obs.: Se o arquivo de pontos a transformar contiverem pontos com os mesmo valores, então no arquivo de saída estes pontos serão identificados com um * antes do valor de identificação (ID).
