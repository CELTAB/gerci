GERCI - Gerênciador de Coordendas Itaipu

Introdução

O projeto GERCI, em um primeito momento, tem como objetivo oferecer uma plataforma de serviços para a transformação de coordenadas. O celtab, baseado em um algorítmo de transformação já existente, fez a adaptação do código para atuar como um serviço RESTful para realizar as transformações. O celtab também desenvolveu um protótipo WEB para validar a utilização da plataforma, oferecendo uma ferramenta inicial para que usuários pudessem utilizar o serviço.

Plataforma de serviço

Inicialmente, foi apresentado ao celtab um sistema Desktop baseado em tecnologia JAVA que implementa um algoritmo de transformação. Baseado neste mesmo sistema, os pesquisadores do celtab isolaram o algoritmo responsável pela transformação, extraindo apenas o necessário do sistema desktop e que é possível de ser incorporado em outras soluções.
